# Academic Risk Prediction Prototype for Nepali Students

## Project Overview
This prototype applies Machine Learning (Classification + Clustering) to predict academic risk levels of students in Nepal using features like sleep hours, attendance, parental education, etc.

## Files & Folders
- `academic_risk_model_complete.ipynb` → Complete Jupyter Notebook with full analysis, model development, evaluation, and explanations.
- `academic_risk_app.py` → Interactive Web GUI for real-time risk prediction.
- `models/random_forest_model.pkl` → Trained model.
- `outputs/` → Key visualization plots.
